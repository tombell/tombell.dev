Software

I stick to macOS for my operating system, as I've used exclusively Mac products for the last 10 years. It gives me the access to most UNIX tools, while not requiring effort to make look good.

Applications

  • Browser: Google Chrome, the reason for this is that Safari doesn't quite have an ad blocker available that can consistently block ads on YouTube and Twitch.
  • Email: Apple Mail, which is hooked up to my Google Workplace (formly Google for Domains) account.
  • Messaging: Apple Messages and Telegram.
  • Chat: Discord
  • Music: Doppler, the simplest and best native music player I've found on Mac for playing FLAC files.
  • Video: MPV, mostly because of its minimal user interface and ability to play every video format I throw at it.
  • Terminal: Alacritty, which I switched to after using iTerm for a number of years. Alacritty won out over Kitty because it was the lighter of the two, and was quicker to get set up in my preferred way.

Terminal Applications

  • Text editor: Neovim has been my text editor exclusively for the last few years, and previous was Vim. I originally switched to Neovim due to the number of improvements it brought with it. I've since stayed with Neovim due to the great things it has added, such as built in LSP support, and using Lua for scripting and configuration.
  • Terminal multiplexer: Tmux is a recent addition to my workflow in the last couple of years. I used to use a lot of splits and tabs in iTerm, however decided to utilise Tmux more. With switching to Alacritty (which has no tabs or splits), using Tmux allows me to generally stay in a single terminal window.
  • Source control: Git, it's the most ubiquitous source control application right now, and GitHub the standard for hosted repositories.
